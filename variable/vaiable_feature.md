# 등호

* = :  변수 오른 쪽에 있는 값을 왼 쪽에 저장하세요.

* == : 비슷한 값이다.

  ​	비교할 두 값을 같은 유형으로 만들고 계산한다.

* = = =: 같은 값이다.

```javascript
var num = 5;
var str = "5";

console.log(num == str);
console.log(num === str);
```



# Undefined

* 변수를 만들었는데, 연산자를 사용해서 어떠한 값도 입력받거나 할당하지 않았을 때 변수의 값은 undefined이다.

```javascript
var num;
num;
-> undefined
```



# null

* 값을 비워둠

```javascript
var num = null;
num;
-> null
```
